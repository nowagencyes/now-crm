NOW Agency CRM Profesional

Incluye:
- Login con Supabase Auth
- Dashboard
- Clientes
- Facturas con IGIC
- Pagos
- Producción
- Redes Sociales
- Accesos
- Diseño oscuro/naranja con logo NOW

Pasos resumidos:
1. Crear proyecto en supabase.com
2. Ejecutar el archivo supabase.sql en SQL Editor
3. Copiar Project URL y anon public key en config.js
4. Subir todo a GitHub
5. Publicar en Cloudflare Pages
